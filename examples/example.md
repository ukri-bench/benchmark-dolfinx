## Example SLURM configuration and output

This folder contains an example slurm submission script, and job output:

- [`submit.sh`](submit.sh) - a SLURM batch submission script, running the benchmark in comparison mode, then two large runs.
- [`slurm.out`](slurm.out) - the console output.
- [`mat_comp-16.json`](mat_comp-16.json), [`Q3-300M.json`](Q3-300M.json) and [`Q6-500M.json`](Q6-500M.json) - example JSON output.
